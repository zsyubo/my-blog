# sizeof

`sizeof`在C语言的保留字，也可以认为是一种(单目)运算符。`可以获取某个数据类型所占用空间的字节数`。

```c++
	std::cout << sizeof(int) << std::endl;
	getChar();
```

`sizeof()`

```
6A 04                push        4  
```

 这里可以到，`sizeof`直接在编译就处理了。 

​	`getChar();`

```
E8 6A F7 FF FF       call        getChar (0D310B4h)  
```

方法底层就是一个`Call`。
